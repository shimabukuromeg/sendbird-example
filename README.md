# Requirement

- Node.js
- Yarn 

# Installation

```bash
$ yarn install
```

## sendbirdのuseridとappidを環境変数に追加する（sendbirdのダッシュボードから取得）

```bash
$ cp .env.example .env.local
```

# Usage

## 開発用

```bash
$ yarn run dev
```

## lint

```bash
$ yarn run lint
```

## Format
```bash
$ yarn run format
```
